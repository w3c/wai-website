Trusted Tester ACT Implementation

Implementation Summary

Trusted Tester version 5.1 is an Test methodology by US federal government. The test results were last updated 4 May 2023. This ACT implementation tests for WCAG 2.0 Level A, AA. See understanding ACT consistency.

Number of Rules Implemented
Consistent Partially consistent
WCAG 2 rules 6 1
Proposed rules 5 5
