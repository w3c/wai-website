---
title: "For Review: ACT Rules Format 1.0 Working Draft"
date: 2017-09-12
---
<p>An updated Working Draft of <a href=" https://www.w3.org/TR/act-rules-format/">Accessibility Conformance Testing (ACT) Rules Format</a> is available for review. This draft addresses all comments received on the previous draft, and is accompanied by fully- and semi-automated <a href="https://w3c.github.io/wcag-act-rules/">sample ACT Rules</a> that implement this rules format. Please submit any comments, especially comments related to implementation experience, as soon as possible, before <strong>10 October 2017</strong>.</p>
