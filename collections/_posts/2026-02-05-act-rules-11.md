---
title: "ACT Rules Format 1.1 is a Web Standard (W3C Recommendation)"
date: 2026-02-05
---

[Accessibility Conformance Testing (ACT) Rules Format 1.1](https://www.w3.org/TR/act-rules-format/) is now a web standard (W3C Recommendation). ACT Rules Format defines a format for writing accessibility test rules. The test rules can be used for developing automated testing tools and manual testing methodologies. For an introduction to ACT Rules, see [Accessibility Conformance Testing (ACT) Overview](https://www.w3.org/WAI/standards-guidelines/act/).
